Alt Text Procedure in MadCap Flare

Step-by-step instructions on how to add alt text on MadCap Flare.

 

Images:

How to add Alt Text on a New Image in MadCap Flare: 

  1. Click on the Insert tab.

  2. Click Image.

  3. Insert your image.

  4. Type a description that describes your new image in the Alternative Text box. Make sure to make it clear, concise, and add a period at the end. Below is an example of alternative text with the picture in step 2.

  5. Click OK. This will add alternative text in the background for you.

How to add Alt Text on an Image Already in MadCap Flare:

  1. Right click on the image you want to change the alternative text for. Make sure this is in your topic pages and not the images tab.

  2. Click on Image Properties.

  3. Type a description that matches your image in the alternative text box. Make sure to make it clear, concise, and add a period at the end.

  4. Click OK. This will add alternative text in the background for you.

 

Multimedia:

How to add Alt Text on New Multimedia in MadCap Flare: 

  1. Click on the Insert tab.

  2. Click Multimedia.

  3. Insert your multimedia video.

  4. Type a description that describes your new image in the Alternative Text box. Make sure to make it clear, concise, and add a period at the end.

  5. Click OK. This will add alternative text in the background for you.

How to add Alt Text on Multimedia Already in MadCap Flare: 

  1. Right click on the multimedia you want to change your alternative text for.

  2. Click Edit Multimedia.

  3. Type a description that describes your multimedia object in the Alternative Text box. Do not include that it is a link, include why it is important.

  4. Click OK. This will add alternative text in the background for you.

 

Links:

How to add Alt Text on a New Link in MadCap Flare:

  1. Right click and drag the text you want to hyperlink.

  2. Click on the Insert tab.

  3. Click Hyperlink.

  4. Type in the link in the link box.

  5. Type in your alternative text that matches the description of your hyperlink in the Alternative Text box. Do not include that it is a link, include why it is important.

  6. Click OK. This will add alternative text in the background for you.

How to add Alt Text on a Link Already in MadCap Flare:

  1. Right click your hyperlink that you want to change your alternative text for.

  2. Click Edit Hyperlink.

  3. Type a description that describes your hyperlink in the Alternative Text box. Do not include that it is a link, include why it is important.

  4. Click OK. This will add alternative text in the background for you.

 

Decorative:

Decorative Images, Videos, and Hyperlinks: 

For decorative objects, you do not have to add any alternative text since it does not add anything for the reader. This will make it smoother for the person to understand, and a quicker process.
